h1>Olá, este é um exemplo de site para o GitHub

Este site foi criado para demonstrar como usar o GitHub Pages para hospedar um site estático. Você pode personalizar este site adicionando mais conteúdo, como imagens, links e estilos CSS.

Para editar este site, basta fazer um fork deste repositório, fazer as alterações desejadas e, em seguida, criar um pull request para que as alterações sejam revisadas e mescladas ao repositório original.

Obrigado por visitar este site de exemplo!
