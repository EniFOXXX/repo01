h1>Olá, este é um exemplo de site para o GitHub

Este site foi criado para demonstrar como usar o GitHub Pages para hospedar um site estático. Você pode personalizar este site adicionando mais conteúdo, como imagens, links e estilos CSS.

Para editar este site, basta fazer um fork deste repositório, fazer as alterações desejadas e, em seguida, criar um pull request para que as alterações sejam revisadas e mescladas ao repositório original.

Se você quiser aprender mais sobre como usar o GitHub Pages, consulte a documentação oficial do GitHub Pages.

Divirta-se criando seu próprio site com o GitHub Pages!

Este é apenas um exemplo básico, e você pode expandir este site adicionando mais páginas, usando CSS para estilizar o conteúdo e até mesmo adicionando JavaScript para interatividade. O GitHub Pages é uma ótima maneira de compartilhar seus projetos, portfólios ou qualquer outro tipo de conteúdo na web de forma gratuita.

Obrigado por visitar este site de exemplo!
